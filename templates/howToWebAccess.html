Connecting Ethernet to Athena

Step 1: Gather Necessary Equipment

Make sure you have the following items:

  • Ethernet cable
  • Athena Side Panel Key
  • Power for Computer (optional)

Step 2: Locate the Ethernet Coupler

The coupler is located in the right side panel of the unit.

  • Unlock the panel door using the key

  • Inside you will see blue ethernet cables, and in yours, the cable should be connected with a coupler (connecting 2 separate ethernet cables together)

  • The red indicator in the image represents where the coupler would be
      • After you separate the two:
    • use the bottom cable (one pointing to the top) and plug it into your computer.

Step 3: Computer Configurations & Web Access

Now we have to set the IP address of the Ethernet Connection on the computer in order to access the web UI

  • Go to Settings > Network and Internet Settings > Ethernet

  • Click on Edit on the IP Address Configuration, and you should see the screen show up in the image.

  • Follow the numbered steps in green, and change the values accordingly.
      • When setting IPV4 address, avoid the following:
    • 192.168.88.206
    • 192.168.88.1
    • 192.168.88.80

  • Close Settings, open a browser (Chrome Preferred), and search 192.168.88.206 in the search bar.

Step 4: Web Configuration & Log Download

Once logged in, in Quick Setup, fill out CPID. This should be provided by your OCPP platform. Hit Save.

  • For help with OCPP Setting:
    • URL: This is the URL to connect to the OCPP backend.
    • SSL: If the OCPP platform requires a network certificate in order to access, then Enable.
    • Port: This is the port address of the OCPP backend. 443 is secure (wss, https), 80 is unsecure.
    • Path: This is the path of the OCPP backend.
    • Authorization Key (Password): This is the password.
    • Customer Number: Unless special customer, all default units will be 0.

  • Hit Apply, wait for sucesss message. If pop-up shows "Failed", hit Apply again.

  • Follow the numbered steps in green, and change the values accordingly.

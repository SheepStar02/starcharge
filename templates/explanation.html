DTC Structure Breakdown

DTC Error codes utilizes the following modular structure: 0xAABBCCCC

    AA Represents Faulty Part

  • 0x10: Main Cabinet
  • 0x20: Dispenser
  • 0x30/31: Cooling Liquid
  • 0x40: Reserved
  • 0x50: E-Box
  • 0x60: EV(related to protocol)

    BB Represents Error Level

  • 0x00: Warning - No Impact
  • 0x01: Warning - Lowering Output Power
  • 0x02: Error - Cannot Perform Charge

    CCCC Represents Error Code


STOPREASON Structure Breakdown

StopReason code uses the following 3 segments: 0xAABBCCCC

    AA Represents Error Origin

  • 0x10: Main Cabinet
  • 0x20: Dispenser
  • 0x30/31: Cooling Liquid
  • 0x40: Reserved
  • 0x80: International

    BB Represents Stop Error Origin

  • 0x00: Trigger caused by self (EVSE)
  • 0x01: Other Equipment
  • 0x02: Other Equipment

    CCCC Represents Error Code
